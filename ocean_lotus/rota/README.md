## Rota
[RotaJakiro](https://blog.netlab.360.com/stealth_rotajakiro_backdoor_en/)(Rota) is the Linux implant believed to be leveraged by Ocean Lotus.
This repo contains the code to emulate the Linux implant based on threat reports listed in the references section below along with reverse engineering efforts.
